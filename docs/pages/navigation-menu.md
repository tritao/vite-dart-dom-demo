A navigation menu provides top-level navigation with rich panels (like a “mega menu”).

## Features

- **Roving focus** across triggers (ArrowLeft/ArrowRight).
- **Panels** rendered as popovers anchored to triggers.
- **Hover + click**: hover switches panels; click toggles.
